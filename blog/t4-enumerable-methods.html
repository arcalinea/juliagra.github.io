Using the Enumerable Method Map

July 11, 2015


There are different ways to iterate over an array and perform an operation. Map (also called 'collect') can be used, as well as 'each'. I will focus on the uses of the enumerable method 'map.

Map returns a new array with the results of running an operation on each element. This is useful when you want your method to be non-destructive. The new array is the same size as the old one, but contains the return values that result from executing the code block on the original array. Map can be used on arrays, hashes, and ranges, but always returns an array.

For example, you could use map on an array of strings to return a new array with just the string lengths.

Writing map with an exclamation point as “map!”, will modify the original array, making the method destructive.

A note on syntax for map: Using 'puts' to output the result of map to the screen prints out the result, but the values of the new array will be nil. To actually return new values, leave out the puts in the block.

In contrast to 'map', the 'each' method always returns the original array. It modifies elements within the code block, but will output the original array.
